# t-digest — โครงสร้างข้อมูลสำหรับประมาณค่าควอนไทล์แบบสตรีมมิง

## ภาพรวม

**t-digest** เป็นโครงสร้างข้อมูลเชิงความน่าจะเป็นที่ออกแบบมาเพื่อประมาณค่าควอนไทล์ (เปอร์เซ็นไทล์) จากสตรีมข้อมูลอย่างมีประสิทธิภาพ อัลกอริทึมนี้คิดค้นโดย Ted Dunning สามารถประมวลผลข้อมูลแบบต่อเนื่องทีละรายการโดยไม่จำเป็นต้องเก็บข้อมูลทั้งหมดไว้ในหน่วยความจำ ขณะเดียวกันยังให้ค่าประมาณควอนไทล์ที่แม่นยำ

คุณสมบัติที่โดดเด่นที่สุดของ t-digest คือความแม่นยำสูงเป็นพิเศษที่ส่วนหาง (tail) ของการแจกแจง แม้ว่าความแม่นยำบริเวณค่ามัธยฐานอาจต่ำกว่าเล็กน้อย แต่ในการประมาณค่าควอนไทล์ที่รุนแรง เช่น เปอร์เซ็นไทล์ที่ 99 หรือ 99.9 นั้น t-digest มีความแม่นยำสูงมาก คุณสมบัตินี้มีความสำคัญอย่างยิ่งในการใช้งานจริง เช่น การตรวจสอบ SLA และการวิเคราะห์ค่าความหน่วง

ภายในตัวโครงสร้าง t-digest จะรวมจุดข้อมูลเป็นกลุ่มที่เรียกว่า "เซนทรอยด์ (centroid)" หน่วยความจำที่ใช้ถูกควบคุมโดยพารามิเตอร์การบีบอัด δ ทำงานด้วยหน่วยความจำที่มีขอบเขต O(δ) โดยไม่ขึ้นกับจำนวนข้อมูลทั้งหมด

## คุณสมบัติหลัก

- **การประมวลผลแบบสตรีมมิง / ออนไลน์** — สามารถเพิ่มจุดข้อมูลทีละจุดได้ ไม่จำเป็นต้องเก็บข้อมูลทั้งหมดไว้ในหน่วยความจำ
- **หน่วยความจำมีขอบเขต O(δ)** — ปริมาณหน่วยความจำที่ใช้ขึ้นอยู่กับพารามิเตอร์การบีบอัด δ เท่านั้น ไม่ขึ้นกับปริมาณข้อมูล
- **ความแม่นยำที่ส่วนหาง** — มีความแม่นยำสูงเป็นพิเศษที่ปลายทั้งสองด้านของการแจกแจง (ใกล้ 0% และ 100%)
- **รวมกันได้** — สามารถรวม t-digest หลายตัวเข้าด้วยกัน เหมาะสำหรับระบบแบบกระจาย

## แนวคิดฟังก์ชันมาตราส่วน

หัวใจของ t-digest คือ "ฟังก์ชันมาตราส่วน (scale function)" ฟังก์ชันนี้ควบคุมว่าเซนทรอยด์สามารถเติบโตได้ใหญ่แค่ไหนในแต่ละตำแหน่งของการแจกแจง บริเวณกลางของการแจกแจงจะอนุญาตให้เซนทรอยด์มีขนาดใหญ่ (เก็บจุดข้อมูลได้มาก) ส่วนบริเวณหางจะอนุญาตเฉพาะเซนทรอยด์ขนาดเล็ก ด้วยกลไกนี้ บริเวณหางจะมีเซนทรอยด์จำนวนมากกว่า จึงทำให้ความแม่นยำในการประมาณค่าที่ส่วนหางสูงขึ้น

## ตัวอย่างการใช้งาน (รหัสเทียม)

```
# สร้าง t-digest (พารามิเตอร์การบีบอัด δ = 100)
td = TDigest.new(delta: 100)

# เพิ่มจุดข้อมูลทีละจุด
td.add(1.0)
td.add(2.5)
td.add(3.7)
td.add(100.0)
td.add(0.01)

# สามารถประมวลผลข้อมูลจำนวนมากได้
for value in data_stream:
    td.add(value)

# สอบถามค่าควอนไทล์
median    = td.quantile(0.5)    # ค่ามัธยฐาน
p99       = td.quantile(0.99)   # เปอร์เซ็นไทล์ที่ 99
p999      = td.quantile(0.999)  # เปอร์เซ็นไทล์ที่ 99.9

# สอบถามย้อนกลับ: หาค่า CDF ของค่าที่กำหนด
cdf_value = td.cdf(42.0)        # สัดส่วนข้อมูลที่น้อยกว่าหรือเท่ากับ 42.0
```
