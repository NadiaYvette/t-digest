# t-digest — بہتے ہوئے ڈیٹا کے لیے مقداری تخمینہ

## خلاصہ

**t-digest** ایک احتمالاتی شماریاتی ساخت ہے جو ڈیٹا کے بہاؤ سے مقداریوں (quantiles) کا مؤثر تخمینہ لگاتی ہے۔ اسے Ted Dunning نے تیار کیا تھا۔ جب کروڑوں قدریں آ رہی ہوں اور آپ وسطانیہ، ۹۹واں صدکیہ یا کوئی اور مقداری جاننا چاہیں، تو تمام قدروں کو حافظے میں رکھنا ممکن نہیں۔ t-digest اس مسئلے کا حل ہے — یہ محدود حافظے میں چلتا ہے اور پھر بھی انتہائی درست نتائج دیتا ہے۔

t-digest کی خاص بات یہ ہے کہ یہ تقسیم کے سروں (tails) پر سب سے زیادہ درست ہوتا ہے۔ یعنی ۹۹ویں یا ۹۹.۹ویں صدکیے کا تخمینہ درمیانی حصے کی نسبت زیادہ قابلِ اعتماد ہوتا ہے۔ یہ خصوصیت تاخیر کی نگرانی، غیر معمولی واقعات کی نشاندہی اور سروس لیول معاہدوں (SLA) کی پیمائش کے لیے نہایت مفید ہے۔

اس منصوبے میں t-digest 28 پروگرامنگ زبانوں میں دستیاب ہے۔

## اہم خصوصیات

- **بہتا ہوا/آن لائن حساب** — قدریں ایک ایک کر کے یا جھنڈوں میں شامل کی جا سکتی ہیں؛ سارا ڈیٹا بیک وقت موجود ہونا ضروری نہیں
- **محدود حافظہ O(δ)** — حافظے کا استعمال ڈیٹا کی مقدار پر نہیں بلکہ سکڑاؤ پیمانے δ (ڈیلٹا) پر منحصر ہے؛ عام طور پر δ=100 کافی ہے
- **سرا-درستگی** — تقسیم کے آخری سروں (۰.۰۱واں یا ۹۹.۹۹واں صدکیہ) پر درستگی بہترین ہوتی ہے، جو عملی استعمال میں سب سے اہم ہے
- **ملانے کے قابل** — دو t-digest ساختوں کو ایک میں ملایا جا سکتا ہے، جو تقسیم شدہ نظاموں کے لیے مثالی ہے

## پیمانہ تفاعل کا تصور

t-digest کہاں زیادہ درستگی رکھنی ہے اور کہاں کم، یہ طے کرنے کے لیے **پیمانہ تفاعل** (scale function) استعمال کرتا ہے۔ ایک پیمائشی پٹی کا تصور کریں جس کے دونوں سروں پر نشانیاں بہت قریب قریب ہیں اور بیچ میں دور دور ہیں۔ پیمانہ تفاعل بالکل یہی کرتا ہے — تقسیم کے سروں پر چھوٹے چھوٹے جھرمٹ (centroids) بناتا ہے تاکہ وہاں درستگی زیادہ ہو، اور درمیان میں بڑے جھرمٹ بناتا ہے جہاں تھوڑی سی غلطی قابلِ قبول ہے۔

## مثال: فرضی ضابطہ (pseudocode)

```
# t-digest بنائیں (سکڑاؤ پیمانہ δ = 100)
td = TDigest.new(delta: 100)

# قدریں شامل کریں
for value in data_stream:
    td.add(value)

# مقداریاں پوچھیں
median    = td.quantile(0.5)    # وسطانیہ
p95       = td.quantile(0.95)   # ۹۵واں صدکیہ
p99       = td.quantile(0.99)   # ۹۹واں صدکیہ
p999      = td.quantile(0.999)  # ۹۹.۹واں صدکیہ

# کسی قدر کا صدکیہ درجہ معلوم کریں
rank = td.cdf(150.0)  # ۱۵۰ سے کم قدروں کا تناسب
```
