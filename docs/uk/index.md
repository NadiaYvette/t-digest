# t-digest — Онлайн-оцінка квантилів

## Що таке t-digest?

**t-digest** — це компактна структура даних для оцінки квантилів та перцентилів із потоку даних. Її розробив Тед Даннінг, і вона особливо корисна у сценаріях, де величезні обсяги даних потрібно обробляти без збереження всіх значень у пам'яті. Замість зберігання кожної окремої точки даних, t-digest підсумовує розподіл в обмеженій кількості так званих центроїдів.

Те, що вирізняє t-digest, — це його чудова точність у хвостах розподілу. Тоді як багато наближених методів втрачають точність саме на екстремальних перцентилях (наприклад, 99,9-й перцентиль), t-digest надає там найкращі результати. Це робить його ідеальним для моніторингу затримок, метрик SLA та виявлення аномалій.

t-digest працює інкрементально: нові значення можна додавати в будь-який момент без необхідності перераховувати наявний підсумок. Крім того, кілька t-digest-ів можна об'єднати (merge), що дозволяє розподілене обчислення на декількох вузлах.

## Ключові властивості

- **Потокова/онлайн-обробка** — Значення додаються по одному або пакетами, без необхідності тримати весь набір даних у пам'яті.
- **Обмежене споживання пам'яті O(delta)** — Кількість центроїдів контролюється параметром стиснення delta і залишається сталою незалежно від обсягу даних.
- **Висока точність у хвостах** — Похибки оцінки є найменшими на краях розподілу (біля 0 та 1), тобто саме там, де це найважливіше на практиці.

## Концепція функції масштабування

t-digest використовує **функцію масштабування** для визначення максимального розміру центроїда в певній позиції розподілу. Поблизу хвостів (квантилі близькі до 0 або 1) функція масштабування дозволяє лише дуже малі центроїди, що забезпечує високу точність. У середині розподілу центроїди можуть бути більшими, оскільки там прийнятна менша точність. Функція масштабування, що використовується в цьому проєкті, — K_1, заснована на арксинусі: `k(q) = (delta / 2*pi) * arcsin(2*q - 1)`.

## Приклад у псевдокоді

```
# Створення нового t-digest з параметром стиснення 100
td = TDigest(delta=100)

# Додавання значень
для i від 0 до 9999:
    td.add(i / 10000.0)

# Запит квантилів
медіана  = td.quantile(0.5)    # ~0.5
p99      = td.quantile(0.99)   # ~0.99
p999     = td.quantile(0.999)  # ~0.999

# Запит значення CDF (зворотний напрямок)
ранг = td.cdf(0.42)            # ~0.42
```
