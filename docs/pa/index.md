# t-digest — ਵਗਦੇ ਡੇਟਾ ਲਈ ਪਰਿਮਾਣਕ ਅੰਦਾਜ਼ਾ

## ਸਾਰ

**t-digest** ਇੱਕ ਸੰਭਾਵਨਾਤਮਕ ਅੰਕੜਾ ਢਾਂਚਾ ਹੈ ਜੋ ਡੇਟਾ ਵਹਾਅ ਵਿੱਚੋਂ ਪਰਿਮਾਣਕਾਂ (quantiles) ਦਾ ਕੁਸ਼ਲ ਅੰਦਾਜ਼ਾ ਲਗਾਉਂਦਾ ਹੈ। ਇਸ ਨੂੰ Ted Dunning ਨੇ ਵਿਕਸਿਤ ਕੀਤਾ ਸੀ। ਜਦੋਂ ਕਰੋੜਾਂ ਮੁੱਲ ਆ ਰਹੇ ਹੋਣ ਅਤੇ ਤੁਸੀਂ ਵਿਚਕਾਰਲਾ ਮੁੱਲ, ੯੯ਵਾਂ ਸ਼ਤਾਂਸ਼ਕ ਜਾਂ ਕੋਈ ਹੋਰ ਪਰਿਮਾਣਕ ਜਾਣਨਾ ਚਾਹੋ, ਤਾਂ ਸਾਰੇ ਮੁੱਲ ਯਾਦਾਸ਼ਤ ਵਿੱਚ ਰੱਖਣਾ ਸੰਭਵ ਨਹੀਂ। t-digest ਇਸ ਸਮੱਸਿਆ ਦਾ ਹੱਲ ਕਰਦਾ ਹੈ — ਇਹ ਸੀਮਿਤ ਯਾਦਾਸ਼ਤ ਵਿੱਚ ਚੱਲਦਾ ਹੈ ਅਤੇ ਫਿਰ ਵੀ ਬਹੁਤ ਸਟੀਕ ਨਤੀਜੇ ਦਿੰਦਾ ਹੈ।

t-digest ਦੀ ਖ਼ਾਸ ਗੱਲ ਇਹ ਹੈ ਕਿ ਇਹ ਵੰਡ ਦੇ ਸਿਰਿਆਂ (tails) ਉੱਤੇ ਸਭ ਤੋਂ ਵੱਧ ਸਟੀਕ ਹੁੰਦਾ ਹੈ। ਭਾਵ ੯੯ਵੇਂ ਜਾਂ ੯੯.੯ਵੇਂ ਸ਼ਤਾਂਸ਼ਕ ਦਾ ਅੰਦਾਜ਼ਾ ਵਿਚਕਾਰਲੇ ਹਿੱਸੇ ਨਾਲੋਂ ਵਧੇਰੇ ਭਰੋਸੇਯੋਗ ਹੁੰਦਾ ਹੈ। ਇਹ ਗੁਣ ਦੇਰੀ ਨਿਗਰਾਨੀ, ਵਿਸੰਗਤੀ ਖੋਜ ਅਤੇ ਸੇਵਾ-ਪੱਧਰ ਸਮਝੌਤਿਆਂ (SLA) ਦੇ ਮਾਪ ਲਈ ਬਹੁਤ ਲਾਭਦਾਇਕ ਹੈ।

ਇਸ ਪ੍ਰੋਜੈਕਟ ਵਿੱਚ t-digest 28 ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਉਪਲਬਧ ਹੈ।

## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

- **ਵਗਦੀ/ਔਨਲਾਈਨ ਗਣਨਾ** — ਮੁੱਲ ਇੱਕ-ਇੱਕ ਕਰਕੇ ਜਾਂ ਸਮੂਹਾਂ ਵਿੱਚ ਜੋੜੇ ਜਾ ਸਕਦੇ ਹਨ; ਸਾਰਾ ਡੇਟਾ ਇਕੱਠਾ ਹੋਣਾ ਜ਼ਰੂਰੀ ਨਹੀਂ
- **ਸੀਮਿਤ ਯਾਦਾਸ਼ਤ O(δ)** — ਯਾਦਾਸ਼ਤ ਵਰਤੋਂ ਡੇਟਾ ਦੀ ਮਾਤਰਾ ਉੱਤੇ ਨਹੀਂ, ਸਗੋਂ ਸੰਕੁਚਨ ਮਾਪਦੰਡ δ (ਡੈਲਟਾ) ਉੱਤੇ ਨਿਰਭਰ ਕਰਦੀ ਹੈ; ਆਮ ਤੌਰ ਤੇ δ=100 ਕਾਫ਼ੀ ਹੈ
- **ਸਿਰਾ-ਸਟੀਕਤਾ** — ਵੰਡ ਦੇ ਅੰਤਿਮ ਸਿਰਿਆਂ (੦.੦੧ਵਾਂ ਜਾਂ ੯੯.੯੯ਵਾਂ ਸ਼ਤਾਂਸ਼ਕ) ਉੱਤੇ ਸਟੀਕਤਾ ਸਭ ਤੋਂ ਵਧੀਆ ਹੈ, ਜੋ ਵਿਹਾਰਕ ਵਰਤੋਂ ਵਿੱਚ ਸਭ ਤੋਂ ਮਹੱਤਵਪੂਰਨ ਹੈ
- **ਰਲ਼ਾਉਣਯੋਗ** — ਦੋ t-digest ਢਾਂਚਿਆਂ ਨੂੰ ਇੱਕ ਵਿੱਚ ਰਲ਼ਾਇਆ ਜਾ ਸਕਦਾ ਹੈ, ਜੋ ਵੰਡੀਆਂ ਪ੍ਰਣਾਲੀਆਂ ਲਈ ਆਦਰਸ਼ ਹੈ

## ਸਕੇਲ ਫੰਕਸ਼ਨ ਦੀ ਧਾਰਨਾ

t-digest ਕਿੱਥੇ ਵਧੇਰੇ ਸਟੀਕਤਾ ਰੱਖਣੀ ਹੈ ਅਤੇ ਕਿੱਥੇ ਘੱਟ, ਇਹ ਤੈਅ ਕਰਨ ਲਈ **ਸਕੇਲ ਫੰਕਸ਼ਨ** ਵਰਤਦਾ ਹੈ। ਸੋਚੋ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਇੱਕ ਫੁੱਟਾ ਹੈ ਜਿਸ ਦੇ ਦੋਵੇਂ ਸਿਰਿਆਂ ਉੱਤੇ ਨਿਸ਼ਾਨ ਬਹੁਤ ਨੇੜੇ-ਨੇੜੇ ਹਨ ਅਤੇ ਵਿਚਕਾਰ ਦੂਰ-ਦੂਰ ਹਨ। ਸਕੇਲ ਫੰਕਸ਼ਨ ਬਿਲਕੁਲ ਇਹੀ ਕਰਦਾ ਹੈ — ਵੰਡ ਦੇ ਸਿਰਿਆਂ ਉੱਤੇ ਛੋਟੇ-ਛੋਟੇ ਸਮੂਹ (centroids) ਬਣਾਉਂਦਾ ਹੈ ਤਾਂ ਜੋ ਉੱਥੇ ਸਟੀਕਤਾ ਵਧੇਰੇ ਹੋਵੇ, ਅਤੇ ਵਿਚਕਾਰ ਵੱਡੇ ਸਮੂਹ ਬਣਾਉਂਦਾ ਹੈ ਜਿੱਥੇ ਥੋੜ੍ਹੀ ਗ਼ਲਤੀ ਮੰਨਣਯੋਗ ਹੈ।

## ਉਦਾਹਰਨ: ਛਦਮਕੂਟ (pseudocode)

```
# t-digest ਬਣਾਓ (ਸੰਕੁਚਨ ਮਾਪਦੰਡ δ = 100)
td = TDigest.new(delta: 100)

# ਮੁੱਲ ਜੋੜੋ
for value in data_stream:
    td.add(value)

# ਪਰਿਮਾਣਕ ਪੁੱਛੋ
median    = td.quantile(0.5)    # ਵਿਚਕਾਰਲਾ ਮੁੱਲ
p95       = td.quantile(0.95)   # ੯੫ਵਾਂ ਸ਼ਤਾਂਸ਼ਕ
p99       = td.quantile(0.99)   # ੯੯ਵਾਂ ਸ਼ਤਾਂਸ਼ਕ
p999      = td.quantile(0.999)  # ੯੯.੯ਵਾਂ ਸ਼ਤਾਂਸ਼ਕ

# ਕਿਸੇ ਮੁੱਲ ਦਾ ਸ਼ਤਾਂਸ਼ਕ-ਕ੍ਰਮ ਜਾਣੋ
rank = td.cdf(150.0)  # ੧੫੦ ਤੋਂ ਘੱਟ ਮੁੱਲਾਂ ਦਾ ਅਨੁਪਾਤ
```
