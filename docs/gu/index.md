# t-digest — પ્રવાહી ડેટા માટે પરિમાણક અંદાજ

## સારાંશ

**t-digest** એક સંભાવનાત્મક આંકડાકીય સંરચના છે જે ડેટા પ્રવાહમાંથી પરિમાણકો (quantiles) નો કાર્યક્ષમ અંદાજ લગાવે છે। તેને Ted Dunning દ્વારા વિકસાવવામાં આવ્યું હતું। જ્યારે કરોડો મૂલ્યો આવતાં હોય અને તમે મધ્યક, ૯૯મું શતાંશક કે અન્ય કોઈ પરિમાણક જાણવા માગો, ત્યારે બધા મૂલ્યોને સ્મૃતિમાં રાખવું વ્યવહારુ નથી. t-digest આ સમસ્યાનું સમાધાન કરે છે — તે મર્યાદિત સ્મૃતિમાં ચાલે છે અને છતાં અત્યંત ચોક્કસ પરિણામો આપે છે.

t-digest ની ખાસિયત એ છે કે તે વિતરણના છેડાઓ (tails) પર સૌથી વધુ ચોક્કસ હોય છે. એટલે કે ૯૯મા કે ૯૯.૯મા શતાંશકનો અંદાજ મધ્ય ભાગ કરતાં વધુ વિશ્વસનીય હોય છે. આ ગુણ વિલંબ દેખરેખ, વિસંગતતા શોધ અને સેવા-સ્તર કરાર (SLA) માપન માટે ખૂબ ઉપયોગી છે.

આ પ્રોજેક્ટમાં t-digest 28 પ્રોગ્રામિંગ ભાષાઓમાં ઉપલબ્ધ છે: Ruby, Haskell, Common Lisp, Scheme, Standard ML, Ada, Prolog, Mercury, C, C++, Go, Rust, Java, Kotlin, Python, Julia, OCaml, Erlang, Elixir, Fortran, Perl, Lua, R, Zig, Nim, D, C# અને Swift.

## મુખ્ય ગુણધર્મો

- **પ્રવાહી/ઑનલાઇન ગણતરી** — મૂલ્યો એક પછી એક અથવા જૂથમાં ઉમેરી શકાય છે; બધો ડેટા એકસાથે હોવો જરૂરી નથી
- **મર્યાદિત સ્મૃતિ O(δ)** — સ્મૃતિ વપરાશ ડેટાની માત્રા પર નહીં, પરંતુ સંકોચન પ્રાચલ δ (ડેલ્ટા) પર આધાર રાખે છે; સામાન્ય રીતે δ=100 પર્યાપ્ત છે
- **છેડા-ચોક્કસાઈ** — વિતરણના અંતિમ છેડાઓ (૦.૦૧મું કે ૯૯.૯૯મું શતાંશક) પર ચોક્કસાઈ શ્રેષ્ઠ હોય છે, જે વ્યવહારુ ઉપયોગમાં સૌથી મહત્ત્વનું છે
- **વિલીનીય** — બે t-digest સંરચનાઓને એકમાં ભેળવી શકાય છે, જે વિતરિત પ્રણાલીઓ માટે આદર્શ છે

## સ્કેલ ફંક્શનની સંકલ્પના

t-digest ક્યાં વધુ ચોક્કસાઈ રાખવી અને ક્યાં ઓછી, તે નક્કી કરવા માટે **સ્કેલ ફંક્શન** વાપરે છે. કલ્પના કરો કે તમારી પાસે એક માપપટ્ટી છે જેના બંને છેડે નિશાનો ખૂબ નજીક છે અને વચ્ચે દૂર દૂર છે. સ્કેલ ફંક્શન બરાબર આ જ કરે છે — વિતરણના છેડાઓ પર નાના નાના જૂથો (centroids) બનાવે છે જેથી ત્યાં ચોક્કસાઈ વધુ હોય, અને મધ્યમાં મોટા જૂથો બનાવે છે જ્યાં થોડી ભૂલ સ્વીકાર્ય છે.

## ઉદાહરણ: છદ્મકૂટ (pseudocode)

```
# t-digest બનાવો (સંકોચન પ્રાચલ δ = 100)
td = TDigest.new(delta: 100)

# મૂલ્યો ઉમેરો
for value in data_stream:
    td.add(value)

# પરિમાણક પૂછો
median    = td.quantile(0.5)    # મધ્યક
p95       = td.quantile(0.95)   # ૯૫મું શતાંશક
p99       = td.quantile(0.99)   # ૯૯મું શતાંશક
p999      = td.quantile(0.999)  # ૯૯.૯મું શતાંશક

# કોઈ મૂલ્યનો શતાંશક-ક્રમ જાણો
rank = td.cdf(150.0)  # ૧૫૦ કરતાં ઓછા મૂલ્યોનું પ્રમાણ
```
