# t-digest — प्रवाही डेटासाठी परिमाणक अंदाज

## सारांश

**t-digest** ही एक संभाव्यतामूलक आकडेवारी संरचना आहे जी डेटा प्रवाहातून परिमाणकांचा (quantiles) कार्यक्षम अंदाज लावते. ती Ted Dunning यांनी विकसित केली. जेव्हा कोट्यवधी मूल्ये येत असतात आणि तुम्हाला मध्यांक, ९९वा शतांशक किंवा इतर कोणताही परिमाणक जाणून घ्यायचा असतो, तेव्हा सर्व मूल्ये स्मृतीत ठेवणे अव्यवहार्य असते. t-digest ही समस्या सोडवते — ती मर्यादित स्मृतीत चालते आणि तरीही अत्यंत अचूक निकाल देते.

t-digest चे वैशिष्ट्य म्हणजे ती वितरणाच्या टोकांवर (tails) सर्वाधिक अचूक असते. म्हणजेच ९९व्या किंवा ९९.९व्या शतांशकाचा अंदाज मधल्या भागापेक्षा अधिक विश्वसनीय असतो. हा गुणधर्म विलंब निरीक्षण, विसंगती ओळख आणि सेवा-स्तर करार (SLA) मोजमापासाठी अत्यंत उपयुक्त आहे.

या प्रकल्पात t-digest 28 प्रोग्रामिंग भाषांमध्ये उपलब्ध आहे: Ruby, Haskell, Common Lisp, Scheme, Standard ML, Ada, Prolog, Mercury, C, C++, Go, Rust, Java, Kotlin, Python, Julia, OCaml, Erlang, Elixir, Fortran, Perl, Lua, R, Zig, Nim, D, C# आणि Swift.

## प्रमुख गुणधर्म

- **प्रवाही/ऑनलाइन गणना** — मूल्ये एकेक करून किंवा गटागटाने जोडता येतात; सर्व डेटा एकत्र असण्याची गरज नाही
- **मर्यादित स्मृती O(δ)** — स्मृती वापर डेटाच्या प्रमाणावर नाही, तर संकोचन प्राचल δ (डेल्टा) वर अवलंबून असतो; सामान्यतः δ=100 पुरेसे असते
- **टोक-अचूकता** — वितरणाच्या अंतिम टोकांवर (०.०१वा किंवा ९९.९९वा शतांशक) अचूकता सर्वोत्तम असते, जी व्यावहारिक उपयोगात सर्वाधिक महत्त्वाची आहे
- **विलीनीय** — दोन t-digest संरचना एकात विलीन करता येतात, जे वितरित प्रणालींसाठी आदर्श आहे

## स्केल फंक्शनची संकल्पना

t-digest कुठे अधिक अचूकता ठेवायची आणि कुठे कमी, हे ठरवण्यासाठी **स्केल फंक्शन** वापरते. कल्पना करा की तुमच्याकडे एक मापपट्टी आहे ज्याच्या दोन्ही टोकांवर खुणा अगदी जवळजवळ आहेत आणि मध्यभागी दूरदूर आहेत. स्केल फंक्शन नेमके हेच करते — वितरणाच्या टोकांवर लहान लहान गट (centroids) बनवते जेणेकरून तिथे अचूकता अधिक राहील, आणि मध्यभागी मोठे गट बनवते जिथे थोडी चूक स्वीकार्य आहे.

## उदाहरण: छद्मकूट (pseudocode)

```
# t-digest तयार करा (संकोचन प्राचल δ = 100)
td = TDigest.new(delta: 100)

# मूल्ये जोडा
for value in data_stream:
    td.add(value)

# परिमाणक विचारा
median    = td.quantile(0.5)    # मध्यांक
p95       = td.quantile(0.95)   # ९५वा शतांशक
p99       = td.quantile(0.99)   # ९९वा शतांशक
p999      = td.quantile(0.999)  # ९९.९वा शतांशक

# एखाद्या मूल्याचा शतांशक-क्रम जाणा
rank = td.cdf(150.0)  # १५० पेक्षा कमी मूल्यांचे प्रमाण
```
