# t-digest — Онлайн-оценка квантилей

## Что такое t-digest?

**t-digest** — это компактная структура данных для оценки квантилей и процентилей из потока данных. Она была разработана Тедом Даннингом и особенно полезна в сценариях, где огромные объёмы данных необходимо обрабатывать без хранения всех значений в памяти. Вместо сохранения каждой отдельной точки данных t-digest суммирует распределение в ограниченном количестве так называемых центроидов.

Отличительная черта t-digest — превосходная точность в хвостах распределения. В то время как многие приближённые методы теряют точность именно на экстремальных процентилях (например, 99,9-й процентиль), t-digest обеспечивает там наилучшие результаты. Это делает его идеальным инструментом для мониторинга задержек, метрик SLA и обнаружения аномалий.

t-digest работает инкрементально: новые значения могут быть добавлены в любой момент без необходимости пересчитывать существующую сводку. Кроме того, несколько t-digest-ов можно объединить (merge), что позволяет осуществлять распределённые вычисления на нескольких узлах.

## Ключевые свойства

- **Потоковая/онлайн-обработка** — Значения добавляются по одному или пакетами, без необходимости хранить весь набор данных в памяти.
- **Ограниченное потребление памяти O(delta)** — Количество центроидов контролируется параметром сжатия delta и остаётся постоянным независимо от объёма данных.
- **Высокая точность в хвостах** — Ошибки оценки минимальны на краях распределения (вблизи 0 и 1), то есть именно там, где это наиболее важно на практике.

## Концепция функции масштабирования

t-digest использует **функцию масштабирования** для определения максимального размера центроида в заданной позиции распределения. Вблизи хвостов (квантили, близкие к 0 или 1) функция масштабирования допускает лишь очень малые центроиды, что обеспечивает высокую точность. В середине распределения центроиды могут быть больше, так как там допустима меньшая точность. Функция масштабирования, используемая в этом проекте, — K_1, основанная на арксинусе: `k(q) = (delta / 2*pi) * arcsin(2*q - 1)`.

## Пример на псевдокоде

```
# Создание нового t-digest с параметром сжатия 100
td = TDigest(delta=100)

# Добавление значений
для i от 0 до 9999:
    td.add(i / 10000.0)

# Запрос квантилей
медиана  = td.quantile(0.5)    # ~0.5
p99      = td.quantile(0.99)   # ~0.99
p999     = td.quantile(0.999)  # ~0.999

# Запрос значения CDF (обратное направление)
ранг = td.cdf(0.42)            # ~0.42
```
