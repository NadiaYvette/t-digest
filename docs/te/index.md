# t-digest — ప్రవాహ డేటా కోసం పరిమాణక అంచనా

## సారాంశం

**t-digest** అనేది డేటా ప్రవాహం నుండి పరిమాణకాలను (quantiles) సమర్థంగా అంచనా వేసే ఒక సంభావ్యతా గణాంక నిర్మాణం. దీన్ని Ted Dunning అభివృద్ధి చేశారు. కోట్ల విలువలు వస్తున్నప్పుడు మధ్యగతం, 99వ శతాంశకం లేదా ఏదైనా ఇతర పరిమాణకం తెలుసుకోవాలనుకుంటే, అన్ని విలువలను మెమరీలో ఉంచడం సాధ్యం కాదు. t-digest ఈ సమస్యను పరిష్కరిస్తుంది — ఇది పరిమిత మెమరీలో నడుస్తుంది మరియు అయినప్పటికీ అత్యంత ఖచ్చితమైన ఫలితాలు ఇస్తుంది.

t-digest యొక్క ప్రత్యేకత ఏమిటంటే ఇది పంపిణీ యొక్క తోకల (tails) వద్ద అత్యధిక ఖచ్చితత్వం కలిగి ఉంటుంది. అంటే 99వ లేదా 99.9వ శతాంశకం యొక్క అంచనా మధ్య భాగం కంటే ఎక్కువ నమ్మదగినదిగా ఉంటుంది. ఈ లక్షణం జాప్యం పర్యవేక్షణ, అసాధారణత గుర్తింపు మరియు సేవా-స్థాయి ఒప్పందాల (SLA) కొలతకు చాలా ఉపయోగకరం.

ఈ ప్రాజెక్టులో t-digest 28 ప్రోగ్రామింగ్ భాషలలో అందుబాటులో ఉంది: Ruby, Haskell, Common Lisp, Scheme, Standard ML, Ada, Prolog, Mercury, C, C++, Go, Rust, Java, Kotlin, Python, Julia, OCaml, Erlang, Elixir, Fortran, Perl, Lua, R, Zig, Nim, D, C# మరియు Swift.

## ప్రధాన లక్షణాలు

- **ప్రవాహ/ఆన్‌లైన్ గణన** — విలువలను ఒక్కొక్కటిగా లేదా సమూహాలుగా చేర్చవచ్చు; మొత్తం డేటా ఒకేసారి అందుబాటులో ఉండవలసిన అవసరం లేదు
- **పరిమిత మెమరీ O(δ)** — మెమరీ వాడకం డేటా పరిమాణంపై కాకుండా సంపీడన పరామితి δ (డెల్టా) పై ఆధారపడుతుంది; సాధారణంగా δ=100 సరిపోతుంది
- **తోక-ఖచ్చితత్వం** — పంపిణీ యొక్క చివరి తోకల (0.01వ లేదా 99.99వ శతాంశకం) వద్ద ఖచ్చితత్వం అత్యుత్తమంగా ఉంటుంది, ఇది ఆచరణాత్మక అనువర్తనాల్లో అత్యంత ముఖ్యమైనది
- **విలీనం చేయదగినది** — రెండు t-digest నిర్మాణాలను ఒకటిగా కలపవచ్చు, ఇది పంపిణీ వ్యవస్థలకు ఆదర్శం

## స్కేల్ ఫంక్షన్ భావన

t-digest ఎక్కడ ఎక్కువ ఖచ్చితత్వం ఉంచాలి, ఎక్కడ తక్కువగా ఉంచవచ్చు అనేది నిర్ణయించడానికి **స్కేల్ ఫంక్షన్** ఉపయోగిస్తుంది. ఒక కొలబద్ద ఊహించుకోండి — దాని రెండు చివర్లలో గుర్తులు చాలా దగ్గరగా ఉంటాయి, మధ్యలో దూరంగా ఉంటాయి. స్కేల్ ఫంక్షన్ సరిగ్గా ఇదే చేస్తుంది — పంపిణీ తోకల వద్ద చిన్న చిన్న సమూహాలు (centroids) ఏర్పరుస్తుంది, అక్కడ ఖచ్చితత్వం ఎక్కువగా ఉంటుంది, మరియు మధ్యలో పెద్ద సమూహాలు ఏర్పరుస్తుంది — అక్కడ కొంచెం తప్పు ఆమోదయోగ్యం.

## ఉదాహరణ: మారు-సంకేతం (pseudocode)

```
# t-digest సృష్టించు (సంపీడన పరామితి δ = 100)
td = TDigest.new(delta: 100)

# విలువలు చేర్చు
for value in data_stream:
    td.add(value)

# పరిమాణకాలు అడుగు
median    = td.quantile(0.5)    # మధ్యగతం
p95       = td.quantile(0.95)   # 95వ శతాంశకం
p99       = td.quantile(0.99)   # 99వ శతాంశకం
p999      = td.quantile(0.999)  # 99.9వ శతాంశకం

# ఒక విలువ యొక్క శతాంశక-క్రమం తెలుసుకో
rank = td.cdf(150.0)  # 150 కంటే తక్కువ విలువల నిష్పత్తి
```
