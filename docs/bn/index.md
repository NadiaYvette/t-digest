# t-digest — প্রবাহী তথ্যের জন্য পরিমাণক আকলন

## সারসংক্ষেপ

**t-digest** একটি সম্ভাব্যতামূলক তথ্য সংরচনা যা তথ্য প্রবাহ থেকে পরিমাণকের (quantile) দক্ষ আকলন করে। এটি Ted Dunning তৈরি করেছিলেন। যখন কোটি কোটি মান আসতে থাকে এবং আপনি মধ্যমা, ৯৯তম শতাংশক বা অন্য কোনো পরিমাণক জানতে চান, তখন সমস্ত মান স্মৃতিতে রাখা সম্ভব নয়। t-digest এই সমস্যার সমাধান করে — এটি সীমিত স্মৃতিতে চলে এবং তবুও অত্যন্ত নির্ভুল ফলাফল দেয়।

t-digest-এর বিশেষ বৈশিষ্ট্য হলো এটি বিতরণের প্রান্তগুলিতে (tails) সবচেয়ে বেশি নির্ভুল। অর্থাৎ ৯৯তম বা ৯৯.৯তম শতাংশকের আকলন মধ্যভাগের তুলনায় অধিক নির্ভরযোগ্য। এই গুণটি বিলম্ব পর্যবেক্ষণ, অসঙ্গতি শনাক্তকরণ এবং সেবা-স্তর চুক্তি (SLA) পরিমাপের জন্য অত্যন্ত কার্যকর।

এই প্রকল্পে t-digest 28টি প্রোগ্রামিং ভাষায় বাস্তবায়িত: Ruby, Haskell, Common Lisp, Scheme, Standard ML, Ada, Prolog, Mercury, C, C++, Go, Rust, Java, Kotlin, Python, Julia, OCaml, Erlang, Elixir, Fortran, Perl, Lua, R, Zig, Nim, D, C# এবং Swift।

## প্রধান বৈশিষ্ট্য

- **প্রবাহী/অনলাইন গণনা** — মান একে একে বা দলে দলে যোগ করা যায়; সমস্ত তথ্য একসঙ্গে থাকার প্রয়োজন নেই
- **সীমিত স্মৃতি O(δ)** — স্মৃতি ব্যবহার তথ্যের পরিমাণের উপর নয়, সংকোচন প্রাচল δ (ডেল্টা)-এর উপর নির্ভর করে; সাধারণত δ=100 যথেষ্ট
- **প্রান্ত-নির্ভুলতা** — বিতরণের চরম প্রান্তে (০.০১তম বা ৯৯.৯৯তম শতাংশক) নির্ভুলতা সর্বোত্তম, যা ব্যবহারিক প্রয়োগে সবচেয়ে গুরুত্বপূর্ণ
- **একত্রীকরণযোগ্য** — দুটি t-digest সংরচনা একটিতে মেলানো যায়, যা বিতরিত ব্যবস্থার জন্য আদর্শ

## স্কেল ফাংশনের ধারণা

t-digest কোথায় বেশি নির্ভুলতা রাখবে আর কোথায় কম, তা নির্ধারণের জন্য একটি **স্কেল ফাংশন** ব্যবহার করে। কল্পনা করুন একটি স্কেল যার দুই প্রান্তে দাগ খুব কাছাকাছি এবং মধ্যখানে দূরে দূরে। স্কেল ফাংশন ঠিক এটাই করে — বিতরণের প্রান্তে ছোটো ছোটো দল (centroid) তৈরি করে যাতে সেখানে নির্ভুলতা বেশি থাকে, আর মধ্যভাগে বড়ো দল তৈরি করে যেখানে সামান্য ত্রুটি গ্রহণযোগ্য।

## উদাহরণ: ছদ্মকূট (pseudocode)

```
# t-digest তৈরি করুন (সংকোচন প্রাচল δ = 100)
td = TDigest.new(delta: 100)

# মান যোগ করুন
for value in data_stream:
    td.add(value)

# পরিমাণক জিজ্ঞাসা করুন
median    = td.quantile(0.5)    # মধ্যমা
p95       = td.quantile(0.95)   # ৯৫তম শতাংশক
p99       = td.quantile(0.99)   # ৯৯তম শতাংশক
p999      = td.quantile(0.999)  # ৯৯.৯তম শতাংশক

# কোনো মানের শতাংশক-ক্রম জানুন
rank = td.cdf(150.0)  # ১৫০-এর কম মানের অনুপাত
```
